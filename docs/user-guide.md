# MSBatch 用户指南

> 写给材料/电镜实验人员（无需编程基础）

---

## 这是什么？

你拍了一张 STEM-HAADF 原子像，知道样品含有哪些元素（比如 Li、Co、O），但不知道：
- 这到底是哪种晶体结构？（层状 R-3m？尖晶石 Fd-3m？）
- 这是哪个晶面？（001？104？）

MSBatch 自动帮你解决这个问题。

---

## 它是怎么工作的？

```
你的样品元素 (Li, Co, O)
    ↓
从 Materials Project 数据库检索所有含 Li-Co-O 的结构
    ↓
对每个候选结构，切出不同晶面的原子层
    ↓
用物理模型模拟 STEM-HAADF 图像
    ↓
你把模拟图和实验图放在一起比
    ↓
找到最像的那个 → 确定结构和晶面
```

**为什么是"人眼比对"而不是自动匹配？** 因为模拟图是理想化的——没有缺陷、没有污染、没有电子束损伤。AI 不理解这些真实世界的微妙差异，但你能。所以系统负责生成候选，你负责最终判断。

## 操作步骤

### 第一步：打开程序

双击 `gui_main.py`，或在命令行运行：
```
D:\conda_envs\msbatch\python.exe gui_main.py
```

你会看到左边是设置面板，右边是结果显示区。

### 第二步：输入参数

**必填：元素**
在 "Elements" 框中输入样品含有的元素，用逗号分隔：
```
Li, Co, O
```

**选填：化学计量比**
如果你知道大致比例（通过 EDS 等手段），可以缩小搜索范围：
```
Co:0.15-0.35, O:0.50-0.75
```
格式：`元素:最小值-最大值`，按化学计量比对总元素数归一化。

**选填：晶面**
默认搜索 4 组低指数面：`(001), (100), (110), (111)`。
如果你已经大致知道是哪个面（比如知道是 (104)），可以指定：
```
001, 104
```

### 第三步：检索候选结构

点击 **"Start Retrieval"** 按钮。程序会连接 Materials Project 数据库，检索所有符合条件的晶体结构。

等待几秒到几十秒（取决于候选数量），右侧会出现候选卡片。

### 第四步：查看候选

每张卡片显示：
- **化学式** — 如 LiCoO₂
- **材料 ID** — mp-xxxxx 格式
- **空间群** — 如 R-3m（层状）、Fd-3m（尖晶石）
- **形成能 / 距 Hull 距离** — 越接近 0 越稳定，排序靠前
- **带隙** — 导体/半导体/绝缘体
- **3D 结构缩略图** — 点击弹出可旋转的晶体结构查看器

每张卡片右上角有勾选框，默认全选。取消不需要的结构（比如你确定样品不是尖晶石相的）。

### 第五步：生成模拟图

点击 **"Start Simulation"**。

程序会自动：
1. 对每个选中的结构，切出指定晶面的原子层
2. 用 abTEM 引擎模拟 STEM-HAADF 图像
3. 保存为 PNG 图片

进度条会显示当前状态（如 "Simulating mp-12345 (001) 3/12"）。

**耗时估计**：每个结构 × 每个晶面约 30 秒 ~ 2 分钟（取决于 slab 大小和 CPU 性能）。

### 第六步：比对实验图

模拟完成后，切换到 **"Simulated Images"** 标签页查看所有模拟图。

点击任意模拟图可放大查看。

在左侧面板底部，点击 **"Upload Image"** 上传你的实验 STEM 图。

切换到 **"Comparison Report"** 标签页：

| 左 | 右 |
|---|---|
| 你的实验图 | 候选模拟图网格 |

并排比对，找出原子排列最相似的。

### 第七步：导出报告

点击 **"Export HTML Report"**，保存为 HTML 文件。

这个文件可以在任何浏览器中打开，包含所有候选信息和模拟图，方便分享给同事。

## 高级设置

点击侧边栏底部的 **"Advanced Settings"** 按钮。

| 参数组 | 说明 | 默认值 |
|--------|------|--------|
| Slab | 板厚 12Å、真空层 15Å | 通常不需要改 |
| 电子光学 | 加速电压 200 kV、会聚角 22 mrad | 按你的电镜参数调整 |
| HAADF 探测器 | 内角 60 mrad、外角 200 mrad | 按探测器几何调整 |
| TDS | Frozen Phonon 15 组态 | 越大越真实但越慢 |

修改后点 **"Save Settings"** 生效。

## 典型使用场景

### 场景 1：已知 LiCoO₂，不确定是层状还是尖晶石

```
元素: Li, Co, O
计量比: Co:0.15-0.35, O:0.50-0.75
晶面: 默认 4 组
→ 检索到层状 (R-3m) 和尖晶石 (Fd-3m) 两种 LiCoO₂
→ 模拟 (001) (100) (110) (111) 四组面
→ 与实验图比对
```

### 场景 2：V₂O₅ 未知晶面

```
元素: V, O
计量比: V:0.20-0.30, O:0.70-0.80
→ 检索到正交相 V₂O₅ (Pmmn)、单斜相 VO₂ 等
→ 分别模拟 → 比对
```

### 场景 3：只想知道可能是什么

```
元素: Ti, O
不填计量比
最大候选数: 100
→ 检索所有 Ti-O 化合物
→ 看哪几个的模拟图像你的实验图
```

## 解读结果

### 模拟图 vs 实验图的差异是正常的

模拟图是**理想完美晶体**，实验图可能有：
- 表面污染 / 非晶层
- 电子束损伤
- 样品倾斜
- 缺陷（空位、位错）
- 探测器噪声

所以不需要完全一致，关注**原子列的相对位置和间距**。

### 如果都不像？

1. 样品可能不是纯相（混合相）
2. 可能是表面重构
3. 可能需要调整电镜模拟参数（欠焦量、像散等）
4. 可能不是晶体材料（非晶）

### 能量距 Hull 的含义

- `energy_above_hull = 0` → 最稳定的相
- `< 0.05 eV/atom` → 亚稳相，可能存在
- `> 0.1 eV/atom` → 不太稳定，室温下可能不存在

优先怀疑 hull 距离小的相。

## 常见问题

**Q: 为什么 O、Li 在模拟图里几乎看不见？**
A: HAADF 是 Z 衬度像——原子越重越亮。Co (Z=27) 比 O (Z=8) 亮 ~10 倍，比 Li (Z=3) 亮 ~50 倍。真实电镜也是这样的。

**Q: 报错 "No candidates found"？**
A: 检查元素拼写（英文）、检查网络连接、确认 API Key 有效。

**Q: 模拟太慢？**
A: 高级设置中降低 Frozen Phonon 组态数（15→5），或减少晶面数量。

**Q: 能分析非晶或缺陷吗？**
A: 当前版本仅支持完整晶体。非晶/缺陷需要后续开发。
